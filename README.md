# 포트폴리오 사이트입니다.


## Pisces

- 프로젝트 이름은 별자리 중 물고기자리를 의미합니다.
- 해당 기간 쯤에 본격적으로 포트폴리오를 기획 하고 제작했기 때문입니다.
- 컨셉은 macOS 9을 컨셉으로 하고있습니다.
- Game 폴더는 제작 중에 있습니다. 목표는 phaserJS + webRTC를 사용한 멀티플레이 게임입니다.

## Usage

```shell

yarn or npm install

yarn dev


```
